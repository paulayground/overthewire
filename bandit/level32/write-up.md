# Analysis

ssh 접속하자 마자 `WELCOME TO THE UPPERCASE SHELL` 라는 문구가 출력하며,
입력한 문자들을 모두 대문자로 변경하여 `sh`을 통해 실행하는 프로그램인 것을 확인했다.
환경변수에 등록되어 있는 `PWD`, `HOME` 등은 치환된 결과로 sh를 실행하는 프로그램이였다.
일반적인 명령어의 경우 대문자로 변하여 사용할 수가 없으며, 대문자로 변할 일이 없는 지정된 환경변수를 이용하여 문제를 풀어야한다.

# Exploitation

현재 실행하고 있는 `uppershell` 프로그램이 쉘을 사용하여 명령어를 실행하기 때문에 프로그램 에서 쉘을 나타내는 환경변수인 `$0`를 입력하여 대문자 치환에 적용 받지 않고 실행되는 쉘로 빠져나올 수 있다
쉘로 빠져나온 뒤에 다음 비밀번호가 있는 위치인 `/etc/bandit_pass/bandit33` 을 통해 비밀번호를 획득할 수 있다.
