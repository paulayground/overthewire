# 배운 점

`find`에서 `-exec, -executable`의 옵션들을 확인했다.
`-exec`를 사용할 경우 명령어 사용 후 `{}`에 실행 결과가 들어가게 되는데 끝을 표현하기 위해 `\;`를 표시한다.
`\;` 와 `\+` 는 명령을 실행하는 방식이 다른데, `\;`의 경우 검색된 파일 하나하나마다 명령을 실행하고 다음부분으로 넘어가고, `\+`의 경우 검색된 파일을 모아서 실행하는 차이가 있다.

# 풀이 과정 기록

# 익스플로잇 코드 정리

```bash
# 실행가능한 파일 아니고(! -executable)
# 사이즈 1033byte(-size 1033c)
# 검색된 결과를 기반으로 file 정보를 확인(-exec file {} \;)
find inhere/ ! -executable -size 1033c -exec file {} \+ | grep ASCII
```
