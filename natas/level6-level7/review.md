# 배운 점

LFI(Local File Inclusion) - 서비스를 통해서 서버의 파일을 불러오거나 실행할 수 있게 하는 보안 취약점이다.
php에서 `include($_GET["page"])`과 같이 파라미터를 검증하지 않고 바로 불러오는 경우 공격자의 요청대로 파일을 불러 전달하는 취약점이 발생한다.
