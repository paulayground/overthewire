# 풀이 과정 기록

`print_credentials()`에서 세션에 `admin=1`이 있으면 flag 획득

`username`, `password` 입력하면 `$maxid 640`에서 랜덤 뽑아서 세션 id 만들고 세션 스타트

구조상 소스코드를 통해 깨야되는 부분은 안보인다.

답을 찾기가 어려줘 힌트를 찾아보니 `maxid 640`개 안에서 admin 세션이 있는 얘기였다.

결과는 쉬운데 사실 admin 랜덤한 숫자 안에 존재한다라는 가정을 생각 안하긴 했다.

admin이 소스코드와 같은 로그인을 통해 로그인을 했고, 랜덤한 세션 아이디를 발급 받았다라는 가정을 하지 않았어서 해당 방법을 찾지는 못했다. 또 세션값은 브라우저 종료하거나 시간이 지나면 날라가는 거고 admin도 로그인 할 때마다 다른 세션값을 받을 텐데라는 생각에 그쪽으로는 생각을 못했다.

힌트를 보고 스크립트를 만들어 해결했다.
