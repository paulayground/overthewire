# 풀이 과정 기록

XOR연산에 대해서도 알고도 있었고, 소스코드를 분석하는 것은 그렇게 어렵지 않았다.

하지만 소스코드의 `$outText .= $text[$i] ^ $key[$i % strlen($key)];` 라인의 `$key[$i % strlen($key)]`부분을 보며 키가 알 수 없는 길이를 가지고 반복한다라는 생각 때문에 역연산으로 값을 구한다하더라도 유의미한 값을 구할 수 있을까에 대한 의구심이 들었다.

결국 다른 방법이 있을까 찾아보다가 포기하고, 일단 역연산된 key 값을 확인은 해봐야겠다는 생각이 들어 반대로 연산했더니 4글자가 반복되는 key값이 나왔다.

key값이 나왔을 때도 어디까지의 key값이 맞는 값일지 확신이 안서 초반에는 반복되는 key값을 그냥 사용했다가 flag를 획득할 수 없었는데, 마지막에 더 고민하다가 반복된 글자를 끊어낸 키를 사용하고 나서 해결됐다. 키의 길이에 대한 생각때문에 너무 오래 걸렸다.
